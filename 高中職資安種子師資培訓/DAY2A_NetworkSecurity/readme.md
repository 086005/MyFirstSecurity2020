# 3.1.網路概論
```
本課程將介紹
[1]基礎的網路概念:
     包含網路類型、網路拓樸、網路架構、網路傳輸媒介等
[2]網路協定:包括OSI 模型與TCP/IP網路協定
    使同學對網路能有更進一步的認識。

1. 瞭解電腦網路類型及現代相關應用
2. 瞭解訊息傳遞方式並能說明網路協定每一層的功能
3. 瞭解TCP、UDP、IP協定與之特性
4. 瞭解IP、DNS運作原理
```
